# GraphQL Voyager
Explore current Margarita GraphQL schema.

## Demo

[margarita-graphql-voyager.now.sh](https://margarita-graphql-voyager.now.sh/)

## Install

```
yarn install
```

## Run

```
yarn start
```

Open [localhost:3002](http://localhost:3002/) in your browser.

## Build and Deploy to now.sh

```
yarn deploy
```
